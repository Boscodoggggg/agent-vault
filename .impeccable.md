# Design Context

## Target Audience

Developers and technical operators who use Codex, Claude Code, Cursor, Cline, and similar AI coding agents for long-running project work.

## Use Cases

- Recover useful work after switching AI tool accounts.
- Continue a project on a new machine.
- Move context from one agent to another without losing decisions, diffs, or next steps.
- Keep local agent transcripts searchable without uploading them.

## Brand Personality

Trustworthy, local-first, developer-grade, precise, and calm. The interface should feel like infrastructure for serious work rather than a chat novelty app.
